Philippines view PAGASA

The dashed boundary on the map is PAGASA's Area of Responsibility. PAGASA publishes no machine-readable feed, so live bulletin status is not shown here yet (roadmap: GitHub Actions ingest). For systems inside the PAR, check pagasa.dost.gov.ph directly.

Reading this page How it works

JMA tracks tropical depressions before they are named, so systems can appear here days before a Hong Kong signal is possible. Himawari infrared shows cloud tops over the open ocean — organised deep convection east of the Philippines is the earliest visible hint, and it is an observation, not a forecast. HKO remains the only warning authority for Hong Kong.
